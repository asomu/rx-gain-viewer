# 밴드 매핑 설정 - 빠른 시작 가이드

**소요 시간**: 5분
**난이도**: ⭐ 쉬움

---

## 밴드 매핑이란?

측정 파일의 밴드 표기를 현장 표준 표기로 자동 변환하는 기능입니다.

**예시**:
```
B41[CN]  →  34_39+41
B1[B7]   →  1[7]_CA
```

---

## 🚀 3단계로 시작하기

### 1단계: 활성화

메인 윈도우에서:
```
☑ Band Mapping  ← 체크
```

### 2단계: 설정 열기

```
[Configure...] 버튼 클릭
```

### 3단계: 매핑 추가

```
[➕ 추가] 클릭
→ 원본 밴드: B41[CN]
→ 매핑 값: 34_39+41
[저장] 클릭
```

**완료!** 이제 변환 시 자동으로 적용됩니다.

---

## 버튼 기능

### 대화상자 버튼

| 버튼 | 기능 | 사용 시점 |
|-----|------|----------|
| **➕ 추가** | 새 매핑 행 추가 | 매핑 추가할 때 |
| **Del** | 행 삭제 | 잘못 입력했을 때 |
| **적용** | 임시 적용 (파일 저장 안 함) | 테스트할 때 |
| **저장** | 파일 저장 후 닫기 | 최종 확정할 때 |
| **취소** | 변경사항 무시 | 실수했을 때 |

### 메인 윈도우 버튼

| 버튼 | 기능 |
|-----|------|
| **Browse...** | JSON 파일 선택 |
| **Configure...** | 설정 창 열기 |

## 📝 실습 예제

### 예제 1: 기본 매핑

**목표**: 3개 밴드 매핑 만들기

**단계**:
1. Configure 클릭
2. ➕ 추가 3번 클릭
3. 아래 내용 입력:

   | 원본 밴드 | 매핑 값 |
   |----------|---------|
   | B1 | TM01_A |
   | B3 | TM03_A |
   | B7 | TM07_A |

4. 저장 클릭
5. 파일명: `basic_mapping.json`

**결과**: 변환 시 자동으로 TM01_A, TM03_A, TM07_A로 표시됩니다.

---

### 예제 2: 지역 코드 매핑

**목표**: 같은 밴드의 지역별 표기

**입력**:

| 원본 밴드 | 매핑 값 |
|----------|---------|
| B41[CN] | 34_39+41_CN |
| B41[NA] | 34_39+41_US |
| B41[EU] | 34_39+41_EU |

**결과**: 지역별로 다른 매핑 값으로 변환됩니다.

---

### 예제 3: CA 조합

**목표**: Carrier Aggregation 표기 변환

**입력**:

| 원본 밴드 | 매핑 값 |
|----------|---------|
| B1[B7] | CA_1_7 |
| B3[B7] | CA_3_7 |
| B1[B3][B7] | CA_1_3_7 |

## ⚠️ 흔한 실수

### 실수 1: 빈 칸 저장
```
❌ 원본 밴드 비워둠
✅ 반드시 입력
```

### 실수 2: 중복 키
```
❌ B41[CN] 두 번 입력
✅ 한 번만 입력
```

### 실수 3: 체크박스 안 함
```
❌ Band Mapping 체크 안 함
✅ 반드시 체크
```

---

## 🔍 에러 해결

### 분홍색 배경 표시
**의미**: 입력 오류
**해결**: 빈 칸 채우기 또는 중복 제거

### "적용됨" 안 뜸
**확인사항**:
1. ☑ Band Mapping 체크했나?
2. 적용 버튼 눌렀나?
3. 원본 표기가 정확한가?

---

## 💡 팁

### 빠른 입력
- Tab 키로 다음 칸 이동
- Enter 키로 편집 시작/종료

### 테스트 방법
1. **적용** 클릭 (저장 안 함)
2. 변환 실행
3. 결과 확인
4. 만족하면 **저장**

### 백업
- 저장하면 자동으로 `.bak` 파일 생성
- 실수해도 복구 가능
